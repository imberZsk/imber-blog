# Hello World

这是一个支持 MDX 的示例博客页面。

## 什么是 MDX?

MDX 是一种强大的文档格式，它允许你在 Markdown 文档中使用 React 组件。

### 主要特点

- 支持所有 Markdown 语法
- 可以导入和使用 React 组件
- 支持 JSX 语法
- 可以使用 JavaScript 表达式

## 为什么选择 MDX?

MDX 结合了 Markdown 的简洁性和 React 的灵活性，是编写技术文档的理想选择。

### 使用场景

- 技术博客
- 文档网站
- 个人笔记
- 教程编写

## 示例代码

```jsx
import { Button } from './components'

# 欢迎使用 MDX

这是一个包含 React 组件的 MDX 文档。

<Button>点击我</Button>
```
